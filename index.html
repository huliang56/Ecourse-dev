电子书技术简介

客户端阅读器

阅读器负责将书本内容呈现给用户，是电子书与用户交互的介质。

E-course电子书阅读器基于CEF(Chromium Embedded Framework)开发。内核与chrome浏览器一致，使用webkit内核，对HTML5的支持较好。（右图为客户端截图）

所用技术

HTML(HyperText Mark-up Language)，是一种标记语言，用于描述内容。

CSS(Cascading Style Sheets)，用于描述页面的样式。

Javascript是一种在浏览器端运行的脚本语言，主要是负责页面的展示和与后台服务器数据的交互。

HTML5在HTML基础上新增了很多特性：

多媒体特性：多媒体特性：包括对视频，音频的支持（之前HTML要播放视频或音频只能安装插件或通过flash实现，但仅支持特定的视频和音频格式）

Web SQL Databases：使用js建立数据库（数据库文件存储在浏览器的缓存目录中），使用sql语句操作数据库（适用的场景：频繁查询数据，数据对安全性不敏感，数据更新较少的场合，这种情况下，为了加快页面响应速度或减轻后端服务器压力，可以在页面初始时将后端数据库的数据加载到浏览器中）

离线支持：可以将页面缓存下来，即使在无网络连接的情况下，仍然可以访问基于网页的应用。

Canvas：绘制图形或动画

WEBGL：对于3D的支持，可以通过此技术绘制3D交互动画或游戏。
